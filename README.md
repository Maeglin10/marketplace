# AeviaMarket — Freelance Services Marketplace

Full-stack marketplace connecting clients with freelancers.

## Features

- Service listings with categories, pricing, reviews
- JWT auth with secure cookie sessions
- Order management with status tracking
- Seller profiles and portfolios
- Responsive design

## Stack

Next.js 15 · Prisma · PostgreSQL · TypeScript · Tailwind CSS
